Etiquetas Basicas de HTML

Las Siglas HTML significan HyperText Markup (Lenguaje de marcas de Hipertexto), el hipertexto en una computadora es texto que posee referencias (Hipervinculos, Link o enlaces) a otros textos.

Atributos y Valores

Los atributos de HTML son un modificador de un tipo de elemento de HTML. Un atributo tampoco modifica la funcionalidad por defecto de un tipo de elemento o proporciona funcionalidad a ciertos tipos de elementos incapaces de funcionar correctamente sin ellos
